# Cross Talk SNS集客プロジェクト

## プロジェクト概要

Cross Talk（ランダム通話アプリ）のユーザー獲得を目的としたSNS運用プロジェクトです。
20〜30代女性（アジア圏中心）をターゲットとし、X・TikTok・Instagramの3媒体で展開します。

## サービス概要

| 項目 | 内容 |
|------|------|
| サービス名 | Cross Talk |
| ジャンル | ランダム通話アプリ |
| ターゲット | 20〜30代女性（アジア圏中心） |
| 主な用途 | 英語学習 / 海外友人作り / 旅行前の現地調査 |

## アプリの優位性

- 顔出し不要
- 5分で自動切断（気軽に試せる）
- 厳密なBAN制度（安心・安全な環境）
- 女性限定モード搭載

## SNS運用方針

### X (Twitter)
言語学習・開発裏話を中心としたテキストコミュニティの形成

### TikTok
「5分間の通話疑似体験」をエンタメ化するショート動画コンテンツ

### Instagram
現地のリアルな声から知る旅行・ライフスタイルのビジュアルマガジン

## ファイル使い方ガイドライン

### docs/前提要件の解説書.md
プロジェクトの根拠となる定義書です。ターゲット像・アプリ優位性・各SNS媒体の方向性を記載しています。
コンテンツ制作や施策立案の際に立ち返るリファレンスとして使用してください。

### meetings/template.md & new_meeting.bat
議事録管理の仕組みです。

1. `new_meeting.bat` をダブルクリックして実行する
2. `meetings/20260421.md` のように当日日付のファイルが自動生成される
3. 生成されたファイルを開いて日時・参加者・アジェンダ等を記入する

同じ日付のファイルが既に存在する場合は上書き確認が表示されます。

### management/task_board.md
カンバン方式のタスク管理ファイルです。

| 操作 | 方法 |
|------|------|
| タスク追加 | `- [ ] タスク名 @担当者 期限: YYYY-MM-DD` の形式で各セクションに追記 |
| タスク完了 | `[ ]` → `[x]` に変更し、【完了 (Done)】セクションへ移動 |
| 担当者変更 | `@名前` の部分を書き換える |

### management/kpi_tracker.csv
X・TikTok・Instagramの数値を週次または月次で記録するCSVファイルです。

記録できる指標の例:
- フォロワー数 / フォロー解除数
- 投稿ごとのインプレッション数・再生数
- 保存数・シェア数・プロフィールクリック数

Excelやスプレッドシートで開いてグラフ化することを推奨します。
